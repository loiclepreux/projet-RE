# Projet Resident Evil

Petit projet personnel autour de l’univers Resident Evil.

Le projet est développé en **PHP** avec une structure **MVC** et fonctionne
uniquement en **local** via **XAMPP**.

## Fonctionnalités
- Quiz Resident Evil
- Système de score
- Pages dynamiques
- Structure MVC (controllers / views / core)

## Technologies utilisées
- PHP
- HTML / CSS / JavaScript
- MySQL
- XAMPP
